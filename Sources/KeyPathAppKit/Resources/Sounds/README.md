# Typing Sounds

This directory contains mechanical keyboard sound effects for the typing sounds feature.

## Sound File Naming Convention

Files should be named: `{profile-id}-{down|up}.{mp3|wav}`

## Included Sound Profiles

| Profile | Files | Source |
|---------|-------|--------|
| Cherry MX Blue | `mx-blue-down.mp3`, `mx-blue-up.mp3` | kbsim (Blue) |
| Cherry MX Brown | `mx-brown-down.mp3`, `mx-brown-up.mp3` | kbsim (Brown) |
| Cherry MX Red | `mx-red-down.mp3`, `mx-red-up.mp3` | kbsim (Black) |
| NK Cream | `nk-cream-down.mp3`, `nk-cream-up.mp3` | kbsim (Cream) |
| Bubble Pop | `bubble-pop-down.mp3`, `bubble-pop-up.mp3` | kbsim (Buckling Spring) |
