# Welcome to KeyPath

KeyPath brings the power of [Kanata](https://github.com/jtroo/kanata) to macOS with a beautiful, native interface. No command line required.

## What is KeyPath?

KeyPath is a complete macOS integration layer for Kanata that handles:

- ✅ **Automatic system setup** - Permissions, drivers, and services
- ✅ **Beautiful SwiftUI interface** - Visual key recording and mapping
- ✅ **Hot reload** - Changes apply instantly via TCP
- ✅ **Intelligent troubleshooting** - Automated conflict detection and resolution
- ✅ **Enterprise-grade reliability** - LaunchDaemon architecture with crash recovery

## Key Features

### Visual Recording
Click record, press keys, click save. No configuration files required.

### App-Specific Keymaps
Different keyboard behaviors for different applications. Perfect for vim-style navigation in Safari or custom shortcuts in VS Code.

### Advanced Remapping
Support for tap-hold, tap-dance, layers, macros, and all Kanata features.
