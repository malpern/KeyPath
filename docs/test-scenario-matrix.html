KeyPath Test Scenario Matrix

Snapshot generated November 15, 2025 after the newest installer, watcher, and uninstall coverage push.

Reality check: (1) The CI runner (run-core-tests.sh) still filters to UnitTestSuite unless CI_INTEGRATION_TESTS=true, so developers should enable that flag before touching installer/privileged flows. (2) Installer reports now provide structured success/failure info, the new AdminCommandExecutor harness validates the exact admin scripts, log rotation and helper AppleScript fallbacks are covered, and the launchctl smoke test uses a fake shim to exercise loadService outside test mode. (3) The rest of the matrix remains to guard the documented behaviors before a large refactor.
