# KeyPath Visual Identity

The KeyPath logo and branding as it appears in the application:

```
⌨️  KeyPath
    Keyboard Remapping Made Simple
```

## Logo Details

- **Icon**: System keyboard icon (SF Symbol: "keyboard.fill")
- **Size**: 48pt in the app, with animated scaling from 0.8 to 1.0
- **Colors**: Linear gradient from blue to purple
- **Effects**: 
  - Glow effect with 6pt blur radius
  - Shadow: blue at 30% opacity, 8pt radius, offset (0, 4)
  - Pulsing animation on the glow layer

## Typography

- **App Name**: "KeyPath"
  - Font: System Bold Rounded, 28pt
  - Linear gradient: blue to purple
  - Character-by-character fade-in animation
  
- **Tagline**: "Keyboard Remapping Made Simple"
  - Font: System Medium, 14pt
  - Color: Secondary label color
  - 80% opacity relative to title

## Animation Sequence

1. Keyboard icon scales from 80% to 100% (spring animation)
2. Glow effect fades in after 0.3s
3. Title appears character by character after 0.6s
4. Tagline fades in after title completes

## Usage in Documentation

For markdown documents, use:
- `⌨️ KeyPath` for the title
- Include the tagline as a subtitle
- Use the keyboard emoji as a visual identifier
